# 天枢 Project - 自动材料更换单元控制

## 项目简介

天枢 项目是一个基于 Raspberry Pi Pico 的自动材料更换单元（AMCU）控制系统，专为 3D 打印机设计。该项目实现了与 Bambu Lab 打印机的通信协议，支持多通道材料管理、电机控制、显示屏交互等功能。

## 主要功能

- **多通道材料管理**：支持最多 16 个通道的材料管理
- **BambuBus 通信协议**：实现与 Bambu Lab 打印机的通信
- **OLED 显示屏**：128x64 SSD1306 显示屏，支持图形界面和状态显示
- **电机控制**：支持 16 个电机的控制，实现材料的进料和退料
- **微动开关检测**：自动检测材料状态
- **AS5600 编码器模拟**：模拟磁编码器功能
- **设置存储**：将配置信息存储到 Flash
- **启动图片显示**：支持自定义启动图片显示

## 硬件要求

- **主控板**：Raspberry Pi Pico
- **显示屏**：SSD1306 128x64 OLED 显示屏
- **通信接口**：I2C（显示屏）、UART（BambuBus）
- **微动开关**：用于检测材料状态
- **电机**：用于材料的进料和退料
- **电源**：24转5V 电源

## 软件架构

### 核心模块

1. **AMCU**：自动材料更换单元控制核心
2. **BambuBus**：与打印机的通信协议实现

### 配置说明

项目的主要配置参数位于代码中：

- **通道数量**：默认支持 16 个通道
- **AMS 数量**：可配置为 1-4 个
- **退料时间**：每个通道可单独设置退料时间（3-30 秒）
- **屏幕休眠**：可设置屏幕休眠时间或常亮

## 功能详解

### 材料管理

- **材料状态**：通过微动开关检测材料是否在线（或直接设置在线/离线）
- **材料长度**：记录每个通道的材料使用长度
- **材料颜色**：支持设置材料颜色

### 电机控制

- **进料**：控制电机正向旋转，将材料送入打印机
- **退料**：控制电机反向旋转，将材料从打印机中退出
- **停止**：停止电机转动

### 显示屏界面

- **主界面**：显示当前通道和状态
- **设置菜单**：可设置退料时间、屏幕休眠、电机调试等
- **关于页面**：显示项目版本和相关信息

### 通信协议

- **BambuBus**：实现与 Bambu Lab 打印机的通信协议
- **数据包类型**：支持材料运动、在线检测、NFC 检测等数据包


## 调试功能

- **电机调试**：可手动控制每个电机的运动
- **通道状态设置**：可手动设置每个通道的状态（离线/在线/自动）
- **AMS 数量设置**：可设置 AMS 的数量（1-4）

## 故障排除

### 常见问题

1. **显示屏不显示**：检查 I2C 连接是否正确
2. **电机不转**：检查电机连接和电源
3. **通信失败**：检查 UART 连接和波特率
4. **材料检测异常**：检查微动开关连接

- **SSD1306 显示屏**：SDA -> 4, SCL -> 5, VCC -> 5V, GND -> GND

## 注意事项

1. 请确保电源供应稳定，避免电压波动
